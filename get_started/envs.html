Note

Given that the ROS Gazebo Gym package adheres to the same structure as the openai_ros package developed by the construct, seamless porting of environments from the openai_ros package to the ROS Gazebo Gym package is feasible. Should you be inclined to transition an environment from the openai_ros package to the ROS Gazebo Gym package, we kindly request your review of the Add new environments and Contribute to ros-gazebo-gym sections, followed by the submission of a pull request containing your ported environment.
